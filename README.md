# Game Wiki for Thinking

ゲーム情報Wiki

---

## 起動方法

### フロントエンド
```bash
cd frontend
npm run dev
# http://localhost:5173
```

### バックエンド
```bash
cd backend
mvn spring-boot:run
# http://localhost:8080
```

### データベース
```bash
psql -U postgres -d gamewiki
```

## 実装済み機能

### コンテンツ管理
- [x] ゲーム一覧・詳細・CRUD（画像付き、ドラッグ並び替え）
- [x] アイテム一覧・詳細・CRUD（画像付き、タグ・ゲーム紐付け）
- [x] ボス一覧・詳細・CRUD（画像付き、タグ・ゲーム紐付け）
- [x] NPC 一覧・詳細・CRUD（画像付き、タグ・ゲーム紐付け）

### タグ・フィルター
- [x] タグ管理（ITEM / BOSS / NPC タイプ別、ゲームごと）
- [x] ゲーム・タグ・キーワードによる絞り込み（選択時リアルタイム反映）
- [x] ボス/NPC タグ名をキーワードとした関連アイテム表示

### UI・UX
- [x] 黒赤基調のダークテーマ
- [x] レスポンシブレイアウト（モバイル対応）
- [x] アイテム・ボス・NPC カードのホバープレビュー（説明文/画像拡大）
- [x] ZIP 画像一括取り込み（クライアント側解凍）

### AI 連携
- [x] 画像選択時に Anthropic API でテキスト自動抽出 → 説明欄入力

### ユーザー・権限
- [x] JWT 認証（ログイン / ログアウト）
- [x] ロール管理（ADMIN / USER）
